# 目标 001: 核心仿真引擎

## 状态

✅ 已完成 (2026-05-22)

## 目标描述

实现"空间快照 + 最近邻补位"的最小可运行仿真闭环。

## 验收标准

- [x] grid.py 能生成矩形、圆形、空心圆、自定义点位的布尔网格快照
- [x] drone.py 有位置、距离计算、移动能力
- [x] matcher.py 实现四步算法：(1) 在位检测 (2) 空缺收集 (3) 距离优先贪心分配 (4) 返回结果
- [x] simulation.py 能串联多步仿真，输出每步状态和轨迹数据
- [x] 全部模块有单元测试，覆盖率充足

## 技术决策

- **无动力学模型**：首期无人机瞬时移动，简化验证
- **欧几里得距离**：直观的距离度量
- **贪心分配**：按最近空缺距离排序，每次满足最近的无人机
- **四舍五入取整**：无人机浮点位置匹配到整数像素
